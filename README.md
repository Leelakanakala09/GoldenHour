## Problem Statement
In emergency situations, panic and delayed decisions can cost lives.
Golden Hour provides instant guidance during the critical first hour after injury.

## Scope
Focused on trauma emergencies such as:
- Road accidents
- Heavy bleeding
- Burn injuries
- Unconscious patients

## Features
- Emergency severity classification
- Multiple emergency selection
- Custom symptom input
- Panic Mode for high-stress situations
- Google Maps hospital navigation

## Tech Stack
- Python
- Streamlit
